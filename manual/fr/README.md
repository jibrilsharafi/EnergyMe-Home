# EnergyMe Home : Manuel d'installation

## Avant de commencer : lisez cette page

> *« Vous êtes sur le point de découvrir des choses dont vous ignoriez l'existence derrière vos murs. »*

Bienvenue ! Dans les 15 prochaines minutes, votre maison va cesser d'être une boîte noire. Vous allez voir, pour la première fois, exactement où va votre énergie : quels circuits la consomment pendant la nuit, quels appareils vident discrètement votre portefeuille, quelles lumières vous avez oubliées d'éteindre il y a trois ans.

Certains utilisateurs découvrent que leur frigo consomme plus que leur télévision. D'autres trouvent une charge fantôme qui tourne depuis des années. Quelques-uns découvrent que leurs panneaux solaires produisent moins qu'ils ne le devraient. **Quoi que votre maison cache, EnergyMe est sur le point de le révéler.**

Accrochez-vous. Cela va être intéressant.

---

*EnergyMe - Home* se connecte directement au secteur alternatif (AC) à l'intérieur de votre tableau électrique. Travailler à l'intérieur d'un tableau électrique est **dangereux** si vous ne savez pas ce que vous faites.

| Ce que vous pouvez faire vous-même | Ce qui nécessite un électricien qualifié |
| --- | --- |
| Ouvrir la boîte et identifier les pièces ([§1](01-installation.md#1-whats-in-the-box)) | Toute intervention à l'intérieur du tableau électrique ([§3](01-installation.md#3-electrical-installation)) |
| Configurer le Wi-Fi et l'interface web ([§4](02-setup.md#4-software-configuration)) | Couper le disjoncteur général |
| Associer les circuits aux canaux dans l'interface ([§4.5](02-setup.md#45-configure-each-channel)) | Connecter L / N à l'appareil |
| | Installer les pinces ampèremétriques (TC) autour des fils |

> **⚠ AVERTISSEMENT : Danger électrique**  
> N'ouvrez pas votre tableau électrique avec le disjoncteur général ENCLENCHÉ. La tension du secteur peut tuer. En cas de doute, **arrêtez et appelez un électricien qualifié**. L'installation doit être conforme à **vos codes et règlements électriques locaux**. EnergyMe S.r.l. n'est pas responsable des installations effectuées par du personnel non qualifié ou en violation des codes locaux.
